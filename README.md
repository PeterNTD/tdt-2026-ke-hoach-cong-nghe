# Kế Hoạch Công Nghệ TDT 2026

Repo tĩnh để xuất bản tài liệu kế hoạch công nghệ TDT 2026 qua GitHub Pages.

## Trang xem

- `index.html` — giao diện desktop chính.
- `mobile.html` — giao diện mobile-first.
- `viewer.html` — bản viewer desktop gốc.
- `simple-index.html` — mục lục đơn giản.

## Cấu trúc

- `01-ke-hoach-chinh/` — kế hoạch chính và 6 lĩnh vực trọng tâm.
- `02-du-an-kpi-ngan-sach/` — danh mục dự án, KPI, ngân sách.
- `03-tien-do/` — tiến độ 06–12/2026.
- `04-slide-trinh-bay/` — slide/deck/PDF.
- `05-mindmap-so-do/` — mindmap và sơ đồ.
- `06-tai-lieu-nguon/` — tài liệu nguồn/tham khảo.
- `07-chatbot-notebooklm/` — kế hoạch thử nghiệm chatbot NotebookLM.

## Lưu ý bảo mật

Repo này chỉ nên public nếu các tài liệu bên trong đã được phê duyệt chia sẻ ra internet.
